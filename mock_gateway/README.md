# Mock API Gateway (Plain JavaScript / ESM)

This project is a lightweight mock API Gateway for a frontend.
It simulates several backend microservices with realistic JSON responses
so your frontend can develop and test without the real services.

## Features

- Express + ESM
- CORS + request logging
- Routes for Users, Posts, Feed, Notifications, Messages, and Search
- Simple in-memory mock data (easy to extend)

## Install & Run

```bash
npm install
npm start
